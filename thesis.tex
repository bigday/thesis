% Abstract in English

\thesisabstract{english}{
Depth cameras have become cheap and are \fixme{becoming} ubiquitous. This makes 3D scanning at home possible. Software for scanning static objects is already available. However, reconstructing a moving human is difficult. Previously suggested methods are slow and no implementation has been made public.

We propose a method for generating an avatar that looks similar to the user and can be animated. We also produce an implementation that works in real-time on a high-end home computer and is freely available.
}
